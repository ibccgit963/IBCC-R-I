# R&I Portal

A Laravel-based **Records & Inward (R&I) Management System** developed for **IBCC (Inter Boards Coordination Commission)** to streamline courier tracking, inward/outward records, dispatch logs, officer assignments, and inter-center file transfers.

The system is designed with a secure **Role-Based Access Control (RBAC)** architecture, enabling efficient workflow management across multiple IBCC centers.

---

## Features

- Courier and dispatch tracking
- Inward and outward record management
- Officer assignment workflow
- Inter-center file/document transfer management
- Role-based access control
- Activity and dispatch logs
- Search and filtering functionality
- Secure authentication and authorization
- Responsive web interface
